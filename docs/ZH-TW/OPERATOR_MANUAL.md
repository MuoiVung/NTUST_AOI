# AOI 系統 — 操作員手冊 (Operator Manual)

本指南旨在說明如何使用 AOI 系統人機界面 (HMI) 來監控電路板檢測、查看結果以及管理系統設置。

---

## 1. 看板概覽 (Dashboard Overview)

主看板提供了所有 PCB 檢測運行的即時列表。

### 核心組件：
- **運行列表 (Run List)**：顯示所有近期掃描記錄，包括序列號 (S/N)、板別 ID 和狀態。
- **搜索與篩選**：使用頂部欄位，根據序列號或訂單 ID 查找特定電路板。
- **狀態指示器**：
    - `COMPLETED` (已完成)：該電路板的檢測週期已結束。
    - `PENDING` (待處理)：檢測仍在進行中或正在等待數據。

---

## 2. 查看檢測結果

查看特定電路板的詳細影像：
1. 在 **運行列表** 中找到該電路板。
2. 點擊該行以打開 **影像庫 (Run Gallery)**。
3. 在影像庫中，您將看到來自 **頂部 (Top)** 和 **底部 (Bottom)** 相機的影像。
4. 帶有紅框的影像表示 `FAIL` (不合格)。綠框表示 `PASS` (合格)。
5. 點擊任何影像即可放大，詳細查看缺陷。

---

## 3. 管理系統設置

**設置 (Settings)** 頁面允許您調整系統管理數據的方式。

### 配置選項：
- **數據保留政策 (Data Retention Policy)**：設置影像在本地 IPC 磁碟保留的天數。超過此期限後，影像將自動移動到長期存儲服務器以節省空間。
- **雲端同步重試 (Cloud Sync Retry)**：如果網絡連接中斷，此選項定義系統重試上傳文件的頻率。

> [!IMPORTANT]
> 修改設置後，請務必點擊 **Save (保存)** 按鈕，以確保新參數應用到後端。

---

## 4. 維護與故障排除

- **影像無法載入**：請確保設置中的 `IMAGE_WATCH_DIR` 與相機軟體保存影像的文件夾一致。
- **數據庫連接錯誤**：如果看板顯示連接錯誤，請使用 **System Launcher** 重啟系統。
- **磁碟空間不足**：如果您看到磁碟警告，請檢查 **數據保留政策**，並考慮減少本地存儲影像的天數。

---

## 5. 日常檢查清單
1. 確保 Docker 正在運行（系統托盤顯示綠色圖標）。
2. 通過 `launcher.py` 啟動系統。
3. 確認當新 PCB 通過機台時，**運行列表** 會即時更新。
4. 定期檢查 **設置**，確保同步任務處於活動狀態。
