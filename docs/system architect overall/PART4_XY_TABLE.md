## 2-3. XY Table 組立說明

### 2-3-1. 安裝前確認

1. 確認機架平面無異物。
2. 確認 XY Table 預組狀態完整。
3. 確認 X 軸、Y 軸滑台無碰撞或變形。
4. 確認螺絲、固定座、拖鏈、Sensor 無損壞。
5. 確認安裝面水平。

---

### 2-3-2. XY Table 安裝步驟

1. 將預組好的 XY Table 放置於機架上。
2. 將 XY Table 初步鎖上機架，但先不要完全鎖緊。
3. 將 X 主軸與鋁擠型對齊平行。
4. 將滑軌固定塊與主軸對齊平行。
5. 使用量測儀器確認平行度。
6. 確認平行度後依序鎖固。
7. 手動移動 X/Y 軸，確認滑動順暢。
8. 確認移動過程無干涉、卡滯或異音。
9. 確認拖鏈活動正常，線材無拉扯。

## 2-7. XY Table 操作與調整

### 2-7-1. X/Y 軸手動移動

1. 使用 HMI 或控制介面進行 JOG 操作。
2. 先以低速移動 X 軸與 Y 軸。
3. 確認滑台移動順暢。
4. 確認移動過程無卡滯、異音或拖鏈干涉。
5. 確認移動方向與操作方向一致。

---

### 2-7-2. XY Table 原點復歸

1. 確認 X/Y 軸移動範圍內無異物。
2. 確認 HOME Sensor 指示燈正常。
3. 執行 X 軸回原點。
4. 執行 Y 軸回原點。
5. 確認回原點完成後座標正常。
6. 若回原點失敗，依異常排除章節檢查。

---

### 2-7-3. XY Table 位置校正

1. 將標準工件放置於檢測位置。
2. 確認相機畫面中心與工件位置。
3. 確認 X/Y 座標與實際位置是否一致。
4. 必要時重新校正原點與檢測點。
5. 校正完成後進行試拍與檢測確認。

### 2-8-5. 潤滑保養

1. 保養前需停止設備並關閉電源。
2. 清潔滑軌與螺桿上的舊油污。
3. 清除灰塵、鐵屑與異物。
4. 依設備商建議補充潤滑油或潤滑脂。
5. 潤滑後低速移動 X/Y 軸，使潤滑均勻分布。
6. 潤滑品型號：待補。

## 2-9. XY Table 異常排除

### 2-9-1. X/Y 軸不動

| 可能原因 | 處理方式 |
|---|---|
| 主電源未開啟 | 確認設備主電源與控制電源 |
| Servo ON 未啟用 | 確認驅動器 Servo ON 訊號 |
| AZD-CX 發生 Alarm | 查看驅動器 Alarm code |
| PLC 無脈波輸出 | 檢查 PLC 輸出與程式狀態 |
| 馬達線或 Encoder 線鬆脫 | 檢查馬達與驅動器配線 |
| 機構卡住 | 停機後檢查滑台、螺桿與滑軌 |

---

### 2-9-2. X/Y 軸移動有異音

| 可能原因 | 處理方式 |
|---|---|
| 滑軌缺油 | 清潔滑軌並補充潤滑 |
| 螺桿有異物 | 停機後清除異物 |
| 聯軸器鬆動 | 檢查並鎖緊聯軸器 |
| 拖鏈干涉 | 檢查拖鏈與線材路徑 |
| 馬達端異音 | 檢查馬達座、聯軸器與馬達狀態 |

---

### 2-9-3. X/Y 軸定位不準

| 可能原因 | 處理方式 |
|---|---|
| 原點偏移 | 重新執行原點復歸 |
| 螺絲鬆動 | 檢查滑台與固定座螺絲 |
| 聯軸器鬆動 | 檢查聯軸器固定狀態 |
| 潤滑不足 | 清潔並補充潤滑 |
| 速度或加減速設定不當 | 降低速度或延長加減速時間 |

---

### 2-9-4. 無法回原點

| 可能原因 | 處理方式 |
|---|---|
| HOME Sensor 無訊號 | 檢查 Sensor 電源、指示燈與 PLC Input |
| 感應片偏移 | 調整感應片位置 |
| 正負極限接反 | 檢查 Sensor 接線與方向 |
| 回原點方向錯誤 | 檢查控制參數 |
| AZD-CX Alarm | 查看 Alarm code 並排除 |

---

### 2-9-5. Sensor 無訊號

| 可能原因 | 處理方式 |
|---|---|
| Sensor 未通電 | 檢查 Sensor 電源 |
| Sensor 線路鬆脫 | 檢查端子與線材 |
| 感應片未進入檢知範圍 | 調整 Sensor 與感應片位置 |
| PLC Input 異常 | 檢查 PLC 輸入點 |
| Sensor 損壞 | 更換同型號 Sensor 測試 |

5. 放入標準工件確認通過間隙。
6. 確認工件輸送時不會卡住或偏移。
7. 鎖固軌道固定螺絲。
